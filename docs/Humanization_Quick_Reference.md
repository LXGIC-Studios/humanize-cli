# Humanization Quick Reference
## The 5-Minute AI Detection Killer

### 🚨 Emergency AI-Killer Checklist (Under 10 minutes)

**If you only have time for 5 changes, make these:**

1. **Add contractions everywhere** (don't, can't, won't, I'm, it's)
2. **Insert personal opinion words** ("honestly," "frankly," "look")
3. **Break one paragraph into uneven pieces** (1 sentence, then 5 sentences)
4. **Add one self-interruption** ("Actually, scratch that...")
5. **Replace one AI word:** delve→dig into, leverage→use, comprehensive→full

### 💀 Instant Death Words (Replace immediately)
- **delve** → dig into, explore, dive into
- **leverage** → use, tap into, work with  
- **comprehensive** → complete, full, detailed
- **robust** → strong, solid, reliable
- **seamless** → smooth, easy, effortless
- **utilize** → just say "use"!
- **facilitate** → help, enable, make easier

### ✅ Human Voice Injectors (Copy/paste ready)

**Openings:**
- "Look, I've been thinking about this..."
- "Okay, this might sound weird, but..."
- "Here's the thing that nobody talks about..."

**Transitions:**
- "But wait, there's more to this story..."
- "Here's where it gets interesting..."
- "(Side note: this is important)"

**Doubt markers:**
- "I think," "maybe," "probably," "I'm not entirely sure, but..."

**Course corrections:**
- "Actually, scratch that..."
- "Wait, let me back up for a second"
- "On second thought..."

### 📊 Quick Testing Stack

1. **Read aloud test** (2 minutes) - Does it sound like talking?
2. **GPTZero.me** (Free) - Should be under 70% AI confidence
3. **Hemingway Editor** (Free) - Target grade 8-10

### ⚡ Genre Speed Hacks

**Marketing copy:** Add "I was skeptical too" + personal failure story
**Technical docs:** Add "This confused me at first" + common mistake warning  
**Social media:** Start mid-thought + add strategic typo
**Long articles:** Vary paragraph lengths dramatically + add tangents

### 🎯 Success Signals
- **Green:** Sentences 5-25 words mixed, contractions used, personal opinions present
- **Red:** All sentences 15-20 words, no contractions, perfectly balanced structure
